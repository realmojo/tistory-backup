Web 프로그래밍/안드로이드

2020-04-03 16:46:25

스도쿠를 가볍게 만들고 난 후 뭘 해볼까 고민후에 같은 그림 찾기를 만들어보기 시작했다.

RPG, FPS 같은 게임들은.. 일단 내 취향도 아니면서 unity를 해야할 이유가 없었기 때문에
한 판에 3분이 넘어가지 않는 퍼즐게임이 내게는 중독성이 더 있어서 개발을 해보기로 하였다

 

모든 건 Vue, Javascript 로 만들었음..!..

터치 반응 속도가 네이티브랑 크게 차이가 없는 듯 한데 0.1 초.. 정도 뭔가 다른 느낌이 있다.

 

  • 터치 라이브러리 vue2-hammer
    https://www.npmjs.com/package/vue2-hammer
  • 이미지는 pixabay 를 이용했고 퍼블릭 도메인 기준으로 저작권 없는 이미지만 사용

 

기획 단계

  • 가볍게 플레이 할 수 있는 방안을 모색 후 개발( 클릭, 진동, 사운드, 이미지 )
  • 좀 더 어렵게 할 수 있는 난이도를 적용
  • 플레이 했던 내용을 토대로 메인페이지를 간단하게 적용
  • 반응형 디자인을 적용해서 (PC, Mobile, Tablet) 호환 할 수 있게 적용
  • 여러번의 테스트 후에 빌드 후 배포

 

개발 자체는 몇시간도(?) 걸리지 않았는데 이미지를 수집하고 가공하는데 꽤 오랜시간이 걸렸었다.

 

 

일단 같은그림찾기는 간단하다!

같은그림을 동시에 2개만 찾으면 되는 거고, 시간 기준이 있는 것과 없는 것으로 구분해서 나누고
이미지 2개를 틀리게 찾으면 다시 원점!!...

 

 

 

만들고 나서 사람들이 조금씩 해준 덕분에 한 달 정도만에 드디어 플레이스토어에서 [같은그림찾기]로 검색을 하면 검색이 되기 시작했다!!! 감격 ㅜㅜ
