들어가기 앞서

일반적으로는 좋은 콘텐츠, 정보들의 글을 작성하여 사람들이 우연히 방문하여 유용한 사이트라고 판별이 될 시에 시간이 걸리더라도 검색엔진에 글이 노출이 됩니다. 하지만 처음 시작하거나 좀 더 구체적으로 적용하고 싶으신 분들에게는 이 글이 도움될 것 같습니다. 네이버 웹 마스터 도구에 대해 공유해 드리겠습니다.

 

 

SEO 란?

SEO(Search Engine Optimization, SEO)는 검색엔진 최적화라는 뜻을 가지고 있으며 콘텐츠의 정보와 질을 높여 자신의 웹사이트 및 블로그 혹은 모든 사이트 등의 노출빈도를 높이는 작업입니다. 키워드 광고로 검색을 높이는 것이 아니며 순수하게 사람들이 검색을 해서 유입할 수 있게 최적화하는 것입니다.

 

 

네이버 웹 마스터 도구

네이버 웹 마스터 도구는 네이버 검색에 좀 더 효육적으로 다가가기 위한 툴이며 네이버에서 운영하고 있는 도구입니다. 아래 사이트에 우리의 티스토리 혹은 개인 웹사이트를 등록하고 요구에 맞게 적용시켜 주어야 합니다.

 

 

웹 사이트 등록

네이버 아이디로 로그인을 하고 접속을 합니다. 처음 접속하신 분들은 아무것도 나타나지 않을 것입니다. 아래에 본인의 웹 사이트 주소(ex. http://realmojo.tistory.com)를 적고 등록을 해줍니다.

 

 

 

 

 

사이트 소유확인

등록한 URL이 본인의 것인지 확인하기 위한 검증이 필요합니다. 본인의 웹 사이트를 직접 구축하시고 운영하시는 분이라면 HTML 파일 업로드를 해도 되겠지만 티스토리의 경우에는 카카오에서 서버를 직접 관리해 주기 때문에 파일 업로드하는 부분에 있어서 제한이 있습니다. 

 

그래서 저희는 HTML 태그 방식을 이용해서 검증을 해주도록 하겠습니다.
본인의 content 값에 맞는 걸로 복사를 하고 티스토리 관리자 페이지에서 스킨 편집을 하도록 하겠습니다. 

<meta name="naver-site-verification" content="XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX" />

 

 

티스토리 [스킨 편집 - HTML 편집]

티스토리에 접속하셔서 스킨 편집 -> HTML 편집을 클릭합니다.

 

스킨 편집을 클릭하고 위의 복사한 값을 아래처럼 <head> 사이에 적어 주고 적용 버튼을 눌러 적용합니다.

 

 

 

소유확인 진행

다시 네이버 웹 마스터 도구로 돌아와서 소유확인을 진행합니다. 보안 문자는 가볍게 등록해줍니다.

 

정상적으로 등록이 되면 다음과 같은 알람 문구가 뜹니다. 확인을 누르고 넘어갑니다.

사이트 소유 확인이 완료되었습니다. 소유확인한 사이트 검색노출은 보장하지 않으며, 노출 영역은 로직에 의해 결정됩니다.

즉 이 알람 문구의 뜻은 네이버에서 검색하는 것을 허용해 주겠다는 것이며 당신이 좋은 정보의 질과 콘텐츠를 만들어 나가면서 사람들의 인지도가 쌓이게 되면 네이버의 알고리즘에 의해서 자동 수집을 해주겠다는 의미입니다. 

 

 

사이트 등록 완료

본인의 사이트가 등록이 된 것을 확인할 수 있습니다.

 

 

시작이 반입니다.
이제 네이버 검색 영역에 노출이 될 수 있음을 인지하고 좋은 콘텐츠를 작성해야 합니다.
