구글 깡통계정 만들기 초간단 7단계

IT 활용법/인터넷

2024-07-01 21:10:10

구글 깡통계정을 만드는 과정을 좀 더 자세히 설명드리겠습니다. 깡통계정은 간단히 말해, 최소한의 정보를 입력하여 만드는 구글 계정입니다. 이 계정은 테스트 용도나 일시적인 용도로 사용되며, 개인 정보 보호를 위해 필요한 경우 사용됩니다. 아래 단계별로 구체적인 방법을 안내합니다.

구글 지메일 바로가기

1. 시크릿 모드 또는 프라이빗 모드 사용

시크릿 모드 또는 프라이빗 모드를 사용하면, 브라우저가 쿠키와 히스토리를 저장하지 않으므로 보다 안전하게 계정을 생성할 수 있습니다.

  1. 크롬 브라우저:
    • Ctrl + Shift + N 키를 눌러 시크릿 모드를 엽니다.
    • 또는 브라우저 우측 상단의 세로 점 세 개 아이콘을 클릭하고 '새 시크릿 창'을 선택합니다.
  2. 파이어폭스 브라우저:
    • Ctrl + Shift + P 키를 눌러 프라이빗 모드를 엽니다.
    • 또는 브라우저 우측 상단의 세로 점 세 개 아이콘을 클릭하고 '새 프라이빗 창'을 선택합니다.

2. 구글 계정 생성 페이지로 이동

  1. 구글 계정 생성 페이지: 구글 계정 만들기로 이동합니다.

3. 계정 정보 입력

계정 생성 양식에 필요한 정보를 입력합니다.

  1. 이름:
    • '이름'과 '성' 칸에 깡통 계정임을 알 수 있는 이름을 입력합니다. 예를 들어, 'Test'와 'Account' 또는 '깡통'과 '계정' 등으로 입력합니다.
  2. 사용자 이름:
    • 원하는 사용자 이름을 입력합니다. 예를 들어, 'testaccount123' 또는 'sampledummy123' 등을 입력합니다.
    • 이미 사용 중인 이름일 경우, 다른 이름을 시도해야 합니다. 구글은 중복 사용자 이름을 허용하지 않습니다.
  3. 비밀번호:
    • 보안성이 낮아도 괜찮지만, 기억하기 쉬운 비밀번호를 입력합니다.
    • 비밀번호 확인 칸에도 동일한 비밀번호를 입력합니다.

4. 추가 정보 입력

추가 정보는 선택 사항이며, 깡통 계정의 경우 가능한 최소한으로 입력합니다.

  1. 전화번호:
    • 선택사항입니다. 입력하지 않아도 됩니다. 다만, 나중에 계정 복구를 위해 추가할 수 있습니다.
  2. 현재 이메일 주소:
    • 입력하지 않아도 됩니다. 비워둡니다.
  3. 생년월일:
    • 아무 날짜나 입력합니다. 예를 들어, '1990년 1월 1일' 등으로 입력합니다.
    • 너무 최근의 날짜를 입력하면 구글에서 나이를 의심할 수 있으므로 적절히 과거의 날짜를 선택합니다.
  4. 성별:
    • 선택사항입니다. '입력하지 않음'을 선택할 수 있습니다.

5. 약관 동의

  1. 구글의 서비스 약관과 개인정보처리방침에 동의해야 합니다.
    • 약관을 읽고, 동의 체크박스를 클릭합니다.
    • '다음' 버튼을 클릭합니다.

6. 추가 보안 설정 (선택사항)

계정을 더 안전하게 보호하고 싶다면 전화번호를 추가할 수 있습니다. 그러나 깡통 계정의 경우 보안을 강화할 필요가 없을 수 있습니다.

  1. 보안 강화:
    • 전화번호를 입력하여 2단계 인증을 설정할 수 있습니다.
    • 이 단계는 선택 사항이며, 필요하지 않다면 생략할 수 있습니다.

7. 계정 생성 완료

위의 모든 단계를 완료하면, 구글 깡통계정이 생성됩니다. 이 계정은 다음과 같은 용도로 사용될 수 있습니다:

  • 테스트 목적: 새로운 앱이나 서비스를 테스트할 때.
  • 일시적 사용: 단기적인 프로젝트나 일시적인 작업에 사용할 때.
  • 개인정보 보호: 개인 정보를 최소화하여 사용해야 하는 경우.

이 과정을 통해 쉽게 구글 깡통계정을 만들 수 있습니다. 필요에 따라 깡통 계정을 생성하여 다양한 용도로 활용하시기 바랍니다.
