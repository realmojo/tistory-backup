아이폰 PIP(Picture in Picture) 듀얼스크린 화면 속 화면 분할 유튜브 화면 설정하기

IT 활용법/iOS

2024-06-30 00:58:19

아이폰 사용자라면 유튜브를 보면서 다른 작업을 동시에 하고 싶었던 적이 한두 번이 아닐 것입니다. 특히 iOS 14부터 지원되는 PIP(Picture in Picture) 기능을 활용하면, 유튜브를 작은 화면으로 띄워 놓고 다른 앱을 사용할 수 있습니다. 이 글에서는 아이폰에서 PIP 기능을 이용해 유튜브 화면을 분할 설정하는 방법을 단계별로 안내하겠습니다.

아이폰 PIP 모드 설정방법

1. PIP 기능이란?

PIP는 '화면 속 화면' 기능으로, 비디오를 작은 창으로 띄워놓고 다른 작업을 동시에 할 수 있게 해주는 기능입니다. 이 기능을 사용하면 영상 시청을 하면서도 메모를 하거나, 메시지를 보내거나, 웹 서핑을 할 수 있습니다.

2. iOS 14 이상 업데이트 확인하기

PIP 기능은 iOS 14 이상에서 지원됩니다. 따라서, 아이폰의 소프트웨어 버전이 iOS 14 이상인지 확인하고, 그렇지 않다면 업데이트가 필요합니다.

  • 설정 > 일반 > 소프트웨어 업데이트로 이동하여 최신 iOS 버전을 설치하세요.

3. 유튜브 프리미엄 가입

현재 유튜브의 PIP 기능은 유튜브 프리미엄 회원만 사용할 수 있습니다. 따라서, 이 기능을 사용하려면 유튜브 프리미엄에 가입해야 합니다.

  • 유튜브 앱을 열고, 프로필 사진을 클릭한 후 유튜브 프리미엄을 선택하여 가입 절차를 진행하세요.

4. PIP 모드 활성화하기

  1. 유튜브 앱을 열고 영상을 재생합니다.
  2. 홈 화면으로 나가거나 다른 앱을 실행하여 유튜브 영상을 작은 창으로 띄웁니다.
    • 영상을 보고 있는 상태에서 홈 화면으로 이동하면 자동으로 PIP 모드로 전환됩니다.
    • 영상을 작은 창으로 유지하면서 크기를 조절하거나 위치를 이동할 수 있습니다.

5. PIP 모드에서 유튜브 설정

  1. 작은 창을 드래그하여 원하는 위치에 배치합니다.
  2. 창의 모서리를 드래그하여 크기를 조절합니다.
  3. 영상을 일시정지하거나 닫으려면 창의 오른쪽 상단에 있는 버튼을 사용하세요.
  4. 다른 앱을 사용하면서 유튜브 영상을 계속 시청할 수 있습니다.

6. PIP 기능 사용 시 주의사항

  • PIP 기능은 배터리 소모가 많으므로, 장시간 사용 시 배터리 잔량을 확인하세요.
  • 유튜브 프리미엄 계정을 사용하지 않으면 PIP 기능을 사용할 수 없습니다.
  • 모든 앱에서 PIP 기능을 지원하지 않으므로, 유튜브 외 다른 앱을 사용할 때는 호환성을 확인하세요.

결론

아이폰의 PIP 기능을 활용하면 유튜브 영상을 보면서도 다른 작업을 동시에 수행할 수 있습니다. 유튜브 프리미엄에 가입하고, iOS 14 이상을 사용하는 경우 이 기능을 사용할 수 있습니다. 이 글에서 제공한 단계를 따라 PIP 기능을 설정하고 효율적으로 멀티태스킹을 즐겨보세요.

유튜브 PIP 기능을 활용하여 더 편리한 스마트폰 생활을 즐기시길 바랍니다.

이상으로 아이폰에서 PIP 기능을 이용해 유튜브 화면을 분할 설정하는 방법에 대한 안내를 마치겠습니다. 추가적인 질문이나 도움이 필요하시면 언제든지 댓글로 문의해 주세요.
