Microsoft 오피스 2019 다운로드 무료 설치하는 방법에 대해서 포스팅을 해보려고 합니다. 최근에는 윈도우를 설치하면 가끔 자동으로 MS 오피스 2019 프로그램이 자동으로 설치가 되어 있는 경우도 있지만 본인이 직접 포맷을 하거나 새로운 컴퓨터를 받았거나 컴퓨터를 잘 모르는 이성에게 도움을 주고자 MS 오피스 2019 프로그램을 찾는 경우가 많습니다. 설치방법은 파일만 찾으면 간단하니 MS 오피스 2019 다운로드하는 방법을 알려드리겠습니다.

MS 오피스 2019 다운로드

MS 오피스 2019를 다운로드 하는 가장 쉬운 방법은 공식 홈페이지에서 다운을 하는 것입니다. 토렌트 같은 사이트를 이용하셔도 되지만 거의다 옛날 버전이며 토렌트 파일에 악성 변조 파일이 심어져 있을 수도 있으니 가급적이면 공식 홈페이지에서 다운 받는 것을 추천드립니다. 

우선 설치를 하기전에 확인해야 할 사항이 있습니다. 시스템 요구사항을 충족하는 지 확인을 해야 하는데요. 처음 설치하는 경우 먼저 사전에 설치 단계를 진행해야 합니다. 글 하단의 사이트를 이용하여 확인해 보시길 바랍니다.

MS Office 64 bit 다운

 

MS 오피스 2019 설치 방법

공식 홈페이지에서 다운로드하시거나 위의 MS Office 64 bit 파일을 다운로드해주시면 파일을 관리자 권한으로 실행시켜 주셔야 합니다. 만약 파일이 실행되지 않거나 오류가 있다면 공식 홈페이지의 설치 문제 해결을 직접 참조해보시길 바랍니다. [설치 문제 해결 링크]

파일을 실행시킨 후에 준비 중이라는 말이 뜨고나서 잠시 설치를 위한 구성요소를 다시 다운로드할 때까지 기다려야 합니다. 대략 5분~10분 정도 걸리니 조금만 기다려 주시길 바랍니다.

설치가 완료되면 아래와 같이 완료되었다는 이미지가 나옵니다. 닫기 버튼을 눌러 설치를 종료해줍니다. 다른 하위 버전과 달리 2019 버전은 설치가 자동으로 진행이 되며 설치가 완료되면 아래와 같이 설치가 완료되었다고 친절하게 안내해 줍니다.

설치된 파일을 실행해 보면 다음과 같이 앱 프로그램 형태로 MS office가 하나의 형태로 묶여서 실행이 됩니다. 한 번에 여러 개를 전부 사용할 수 있어서 MS Office 2019 버전은 매우 편리하게 만들어졌습니다.

가정용 비즈니스용 엔터프라이즈용 교육용

MS Office는 어디에서 사용하느냐에 따라서 버전이 4가지로 존재 합니다. 자세한 사항은 공식 홈페이지에 자세히 나와 있으니 연결 링크만 드리겠습니다. 설치를 하는 도중 궁금하신 분들은 한 번 가볍게 읽어 보시길 바랍니다.

가정용 / 비즈니스용 / 엔터프라이즈용 / 교육용

가정용으로 진행하시면 최대 5개의 디바이스에서 사용이 가능하고 비즈니스용은 회사나 기업에서 단체로 사용할 때 적합한 용도 입니다. 엔터프라이즈용은 대기업이나 1000명 이상이 같이 사용해야 하는 경우가 적합하며 교육용은 학생이나 학교에서 사용할 때 좋은 버전입니다.

MS Office 2019 설치 정리

MS Office 2019는 클라우드, 엑셀, 아웃룩, 워드, 파워포인트 등 여러 개를 사용할 수 있으며 라이센스 없이도 실행이 가능합니다. 다만 라이센스를 구입하였을 경우에는 최대 5개의 디바이스에서 연동이 가능합니다.
