윈도우 설치하는 방법 완벽 가이드 (USB 부팅부터 설치, 인증까지 단계별 설명)

IT 활용법/윈도우

2025-04-08 08:54:08

 

윈도우(Windows) 설치는 PC를 포맷하거나 새로 구매한 조립 컴퓨터에 운영체제를 설치할 때 꼭 필요한 작업입니다.
특히 SSD 교체, 포맷, 바이러스 감염, 성능 저하 등의 이유로 윈도우를 새로 설치하고자 하는 분들이 많습니다.

이번 글에서는 2025년 기준으로 윈도우 11/10을 USB로 설치하는 방법을 상세하게 알려드립니다.
USB 부팅디스크 만들기부터 설치, 정품 인증까지 단계별로 차근차근 안내드리니 초보자도 쉽게 따라하실 수 있습니다.

1. 설치 준비물: 윈도우 설치 전 꼭 챙겨야 할 것들

✅ 반드시 준비해야 할 목록

  • 8GB 이상 USB 메모리 (가능하면 16GB 이상)
  • 인터넷 연결된 PC (설치 파일 다운로드용)
  • 윈도우 설치 파일 (공식 Microsoft 사이트)
  • 설치 대상 컴퓨터 (노트북 또는 데스크탑)

? 공식 윈도우 설치 파일 다운로드 주소:
https://www.microsoft.com/ko-kr/software-download/windows11

2. 윈도우 설치 USB 만들기 (미디어 생성 도구 사용)

? USB 부팅 디스크 만드는 순서

  1. 위 공식 링크 접속 후
    [지금 도구 다운로드] 클릭 → MediaCreationTool 다운로드
  2. 다운로드한 .exe 파일 실행
  3. 동의함 선택 후,
    다른 PC용 설치 미디어 만들기 선택
  4. 언어/에디션/아키텍처(64비트) 설정 후 USB 플래시 드라이브 선택
  5. 연결한 USB 선택 → 설치 파일 자동 생성 시작

⚠ 주의: 이 과정에서 USB 안의 모든 데이터는 삭제됩니다. 백업 필수!

3. 컴퓨터에 USB 연결 후 BIOS에서 부팅 순서 설정

? 부팅 순서 변경 방법 (BIOS 진입)

  1. PC 전원을 켜자마자 DEL, F2, F10, F12 중 하나 반복 누름
    (제조사별로 다름, 화면 하단에 표시됨)
  2. BIOS 또는 UEFI 화면 진입 후
    Boot > Boot Priority 또는 Boot Order 메뉴로 이동
  3. USB 드라이브를 최우선 순서로 설정
  4. 저장(Save) 후 재부팅 → 자동으로 윈도우 설치 화면 진입

? 최근 메인보드는 F11(부팅 디바이스 선택) 으로 빠르게 USB 선택 가능

4. 윈도우 설치 과정 (초보자도 따라할 수 있는 단계)

? 설치 과정 단계별 정리

  1. 윈도우 로고 화면 → 언어/시간/키보드 설정 → [다음] 클릭
  2. [지금 설치] 클릭
  3. 제품 키 입력 (건너뛰기 가능, 나중에 인증)
  4. 설치할 에디션 선택 (Windows 11 Home, Pro 등)
  5. 설치 유형: 사용자 지정보기(고급) 선택

파티션 설정

  • 기존 윈도우가 설치된 드라이브 삭제 → Unallocated 상태로 만들기
  • 새로 만들기 → 드라이브 0에 설치
  1. 파일 복사 완료 → 자동 재부팅 → 초기 설정 진입

5. 윈도우 초기 설정 및 정품 인증 방법

? 초기 설정

  • 국가, 키보드 레이아웃 선택
  • 네트워크 연결 (Wi-Fi 또는 랜선)
  • Microsoft 계정 로그인 또는 로컬 계정 생성
  • 개인 설정 및 위치 권한 설정

? 정품 인증

  • 디지털 라이선스 보유자: 인터넷 연결만 해도 자동 인증
  • 제품 키가 있다면:
    설정 > 시스템 > 정품 인증 > 제품 키 변경에서 입력
  • 정품 인증 상태 확인:
    CMD → slmgr /xpr 입력

6. 윈도우 설치 후 필수 작업: 드라이버와 업데이트

✅ 설치 후 꼭 해야 할 것들

  • Windows Update 실행: 최신 보안 업데이트
  • 그래픽카드 드라이버 설치 (NVIDIA, AMD)
  • 메인보드 칩셋 드라이버는 제조사 홈페이지에서 다운로드
  • 필수 프로그램 설치 (브라우저, 압축 프로그램, 오피스 등)

? 유용한 참고 링크

  • Windows 11 공식 다운로드 페이지
  • 윈도우 정품 인증 가이드
  • 부팅 USB 만들기 공식 도움말

✅ 마무리 요약

단계 설명
1단계 USB 8GB 이상 준비 및 설치 도구 다운로드
2단계 MediaCreationTool로 USB 부팅 디스크 만들기
3단계 BIOS 진입 → USB 부팅 순서 설정
4단계 윈도우 설치 진행 (파티션 설정 포함)
5단계 초기 설정 및 정품 인증
6단계 드라이버 및 필수 프로그램 설치

윈도우 설치는 초보자에게는 어렵게 느껴질 수 있지만, 위 순서를 차근차근 따라가면 누구나 성공할 수 있습니다.
혹시 설치 중 에러나 블루스크린이 발생하거나 드라이버 문제 등 어려움이 생기면 언제든지 질문 남겨주세요. ?
