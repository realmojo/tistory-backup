윈도우 하단바 오류 작업표시줄 해결하는 7가지 방법

IT 활용법/윈도우

2024-07-01 03:26:44

윈도우 작업 표시줄이 사라지거나 제대로 작동하지 않는 문제는 꽤 자주 발생하는 문제 중 하나입니다. 다음은 이러한 문제를 해결하는 몇 가지 방법입니다.

방법 1: 작업 표시줄 숨김 설정 확인

작업 표시줄이 자동으로 숨겨지는 설정이 켜져 있을 수 있습니다.

  1. 시작 메뉴를 열고 설정을 클릭합니다.
  2. 개인 설정을 선택합니다.
  3. 작업 표시줄을 선택합니다.
  4. 작업 표시줄 자동 숨기기 옵션이 켜져 있는지 확인하고, 꺼져 있는지 확인합니다.

방법 2: 파일 탐색기 재시작

작업 표시줄이 반응하지 않을 때 파일 탐색기를 재시작하면 문제가 해결될 수 있습니다.

  1. Ctrl + Shift + Esc를 눌러 작업 관리자를 엽니다.
  2. 프로세스 탭에서 Windows 탐색기를 찾습니다.
  3. Windows 탐색기를 선택하고 다시 시작 버튼을 클릭합니다.

방법 3: 작업 표시줄을 재설정

작업 표시줄을 재설정하면 많은 문제를 해결할 수 있습니다.

  1. Ctrl + Shift + Esc를 눌러 작업 관리자를 엽니다.
  2. 파일 메뉴에서 새 작업 실행을 선택합니다.
  3. "powershell"을 입력하고 확인을 클릭합니다.
  4. PowerShell 창에 아래 명령어를 입력하고 Enter를 누릅니다: 
    Get-AppxPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
  5. 프로세스가 완료되면 컴퓨터를 재부팅합니다.

방법 4: 디스플레이 설정 확인

작업 표시줄이 다른 모니터로 이동했을 수 있습니다.

  1. 설정을 엽니다.
  2. 시스템을 선택합니다.
  3. 디스플레이 탭을 선택합니다.
  4. 다중 디스플레이 설정에서 모든 디스플레이가 제대로 감지되는지 확인합니다. 필요하면 디스플레이를 다시 배열합니다.

방법 5: Windows 탐색기 강제 종료 및 다시 시작

작업 표시줄이 반응하지 않을 때 Windows 탐색기를 강제 종료하고 다시 시작하면 문제를 해결할 수 있습니다.

  1. Ctrl + Shift + Esc를 눌러 작업 관리자를 엽니다.
  2. 파일 메뉴에서 새 작업 실행을 선택합니다.
  3. "explorer.exe"를 입력하고 확인을 클릭합니다.

방법 6: 시스템 파일 검사기 실행

시스템 파일 검사기를 사용하여 손상된 시스템 파일을 복구할 수 있습니다.

  1. 시작을 클릭하고 "cmd"를 입력합니다.
  2. 명령 프롬프트를 마우스 오른쪽 버튼으로 클릭하고 관리자 권한으로 실행을 선택합니다.
  3. 명령 프롬프트 창에 아래 명령어를 입력하고 Enter를 누릅니다: 
    sfc /scannow

방법 7: Windows 업데이트 확인

Windows 업데이트를 통해 버그 수정 및 기능 개선을 받을 수 있습니다.

  1. 설정을 엽니다.
  2. 업데이트 및 보안을 선택합니다.
  3. Windows 업데이트를 선택하고 업데이트 확인을 클릭합니다.

이와 같은 방법들을 통해 작업 표시줄 사라짐 문제를 해결할 수 있습니다. 문제가 계속될 경우, Windows 지원 센터나 전문가의 도움을 받아보시기 바랍니다.
