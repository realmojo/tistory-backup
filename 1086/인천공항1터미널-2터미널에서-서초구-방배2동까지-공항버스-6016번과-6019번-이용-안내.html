인천공항에서 서초구 방배2동까지 공항버스 이용 개요

서초구 방배2동 주민들은 인천공항에서 방배2동까지 편리하게 이동할 수 있는 공항버스 6016번과 6019번을 이용할 수 있습니다. 이 두 노선은 방배2동 인근의 주요 정류장을 경유하여 인천공항과 연결되어 있어, 공항에서 방배2동으로의 이동이 용이합니다.

공항버스 6016번 노선 및 운행 정보

  • 노선 경유지: 인천공항2터미널 → 인천공항1터미널 → 효사정앞 → 흑석역 → 이수교(방배본동) → 정금마을(방배경찰서) → 경문고교(배나무골 앞) → 총신대입구역(남성시장입구앞) → 이수역(사당우체국) → 사당역(방배경찰서앞) → 방배래미안아파트 → 임광아파트 → 방배역 → 서울고등학교 → 서초(3동)사거리 → 서울교대
  • 운행 시간: 인천공항2터미널 기준 첫차 06:30, 막차 22:50 / 서울교대 기준 첫차 04:35, 막차 20:00
  • 배차 간격: 약 30~50분 간격으로 하루 25회 운행
  • 소요 시간: 약 75~90분
  • 요금: 성인 17,000원 / 어린이 12,000원

6016번 버스는 방배2동 인근의 방배역과 이수역 등을 경유하여 인천공항과 연결되므로, 방배2동 주민들에게 편리한 선택지입니다.

공항버스 6019번 노선 및 운행 정보

  • 노선 경유지: 인천공항2터미널 → 인천공항1터미널 → 여의도 메리어트호텔 → 지적공사(여의도자이, 샛강역) → 공군호텔 → 서울지방병무청 → 서울공업고등학교 → 신대방삼거리역 → 핸드픽트호텔 → 장승배기역 → 동작보건소 → 상도역 → 숭실대입구역 → 숭실대별관 → 사당해그린 → 사당롯데캐슬골든포레 → 총신대 → 남성역 → 남성초(사당대아신협) → 이수역 → 내방역(방배1, 4동)
  • 운행 시간: 인천공항2터미널 기준 첫차 06:45, 막차 22:45 / 내방역 기준 첫차 04:20, 막차 20:00
  • 배차 간격: 약 30~60분 간격으로 하루 21회 운행
  • 소요 시간: 약 90분
  • 요금: 성인 17,000원 / 어린이 12,000원

6019번 버스는 방배2동 인근의 내방역을 경유하여 인천공항과 연결되므로, 방배2동 주민들에게 또 다른 편리한 선택지입니다.

승차권 구매 및 탑승 안내

  • 공항에서 시내로 이동 시: 인천공항 내 매표소 또는 무인발권기에서 승차권을 구매한 후 탑승
  • 시내에서 공항으로 이동 시: 버스 탑승 시 현금 또는 교통카드(티머니, 후불교통카드)로 결제 가능
  • 주의사항: 신용카드, 기후동행카드, 디스커버서울카드는 사용 불가
