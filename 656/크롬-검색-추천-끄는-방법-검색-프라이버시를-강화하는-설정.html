크롬 검색 추천 끄는 방법 검색 프라이버시를 강화하는 설정

IT 활용법/인터넷

2024-08-08 20:58:48

구글 크롬은 사용자가 검색할 때 검색어에 대한 추천을 제공하여 보다 빠르게 검색할 수 있도록 도와줍니다. 그러나 때때로 이러한 추천 기능이 불편하거나 개인 정보 보호의 측면에서 우려가 될 수 있습니다. 검색 추천을 끄는 방법은 비교적 간단하며, 이를 통해 보다 개인적인 검색 환경을 만들 수 있습니다. 본 글에서는 구글 크롬에서 검색 추천을 비활성화하는 방법을 단계별로 안내합니다.

구글 깡통계정 만들기 초간단 7단계

크롬 검색 추천 끄는 방법

1. 크롬 설정 열기

크롬 다크모드 설정하는 방법 PC, iOS, Android 종합 가이드

크롬 검색 추천을 끄기 위해서는 먼저 크롬의 설정 메뉴에 접근해야 합니다.

  • 설정 열기: 크롬 브라우저를 열고, 오른쪽 상단의 세로 점 3개(메뉴 아이콘)를 클릭합니다.
  • 설정 선택: 드롭다운 메뉴에서 설정을 선택합니다. 크롬 설정 메뉴에서 자세한 내용을 확인할 수 있습니다.

2. 자동 완성 및 검색 추천 비활성화

설정 메뉴에서 자동 완성과 검색 추천 기능을 비활성화할 수 있습니다.

  • 자동 완성 설정 찾기: 설정 페이지에서 검색 엔진 또는 검색 섹션을 찾습니다.
  • 검색 자동 완성 기능 비활성화: 주소 표시줄의 자동 완성 또는 주소 표시줄에서 검색 제안 옵션을 비활성화합니다. 이 옵션을 끄면 검색 창에 검색 추천이 더 이상 나타나지 않습니다.

3. 검색 기록 관리

검색 기록이 검색 추천에 영향을 미칠 수 있으므로, 검색 기록을 관리하는 것도 중요합니다.

  • 검색 기록 삭제: 설정 메뉴에서 개인정보 및 보안을 선택한 후 인터넷 사용 기록 삭제를 클릭합니다. 인터넷 사용 기록 삭제 방법에서 자세한 정보를 확인할 수 있습니다.
  • 기록 삭제 옵션 선택: 기록 삭제에서 기본 탭을 선택하고, 삭제할 데이터의 범위를 설정한 후 데이터 삭제 버튼을 클릭합니다.

4. 크롬 브라우저의 프라이버시 설정 강화

추가적으로, 브라우저의 프라이버시 설정을 강화하여 검색 추천을 줄일 수 있습니다.

  • 쿠키 및 기타 사이트 데이터 관리: 설정 -> 개인정보 및 보안 -> 쿠키 및 기타 사이트 데이터를 선택합니다. 쿠키 설정을 조정하여 개인 정보 보호를 강화할 수 있습니다. 쿠키 관리 방법에서 더 많은 정보를 얻을 수 있습니다.

5. 구글 계정에서 검색 추천 관리

구글 계정 설정에서도 검색 추천을 조정할 수 있습니다.

  • 구글 계정 설정 열기: 구글 계정에 로그인한 후, 구글 계정 관리 페이지로 이동합니다.
  • 활동 제어 설정: 데이터 및 개인 정보 보호 섹션에서 웹 및 앱 활동을 클릭하여 검색 기록의 저장을 중지할 수 있습니다. 이 설정을 비활성화하면 구글이 검색 기록을 기반으로 추천을 제공하지 않습니다. 구글 계정 활동 제어에서 더 많은 정보를 확인할 수 있습니다.

검색 추천 비활성화 후 주의 사항

1. 검색 속도 저하 가능성

검색 추천 기능을 비활성화하면 검색어 자동 완성 기능이 작동하지 않으므로 검색 속도가 약간 저하될 수 있습니다. 그러나 이는 사용자의 편의성을 높이는 대신 프라이버시를 강화하기 위한 조치입니다.

2. 검색 결과 개인화 부족

검색 추천 기능을 끄면, 검색 결과의 개인화가 줄어들 수 있습니다. 추천 기능이 개인화된 검색 결과를 제공하기 때문에, 추천을 끄면 검색 결과가 일반적인 정보로만 제공될 수 있습니다.

3. 클라우드 서비스와의 연동 문제

구글의 클라우드 서비스와 검색 기록 연동이 제한될 수 있습니다. 검색 기록이 클라우드에 저장되지 않으므로, 여러 장치에서 검색 기록을 동기화할 수 없습니다.

결론

크롬에서 검색 추천을 끄는 방법은 설정에서 자동 완성 및 검색 추천 기능을 비활성화하는 것부터 시작합니다. 검색 기록을 삭제하고, 구글 계정에서 검색 활동 제어를 조정하는 것도 도움이 됩니다. 이러한 설정을 통해 검색 프라이버시를 강화하고, 개인적인 검색 환경을 조성할 수 있습니다. 구글 크롬 지원 페이지에서 추가적인 도움을 받을 수 있습니다.
