사이트 최적화

사이트 최적화란 검색엔진이 웹 페이지를 수집하기에 가장 잘맞는 최적의 조건으로 블로그 & 웹 사이트 등을 변경해 주는 작업을 의미합니다. 국내에는 다행히 네이버 웹 마스터 도구가 있어서 편리하게 최적화 작업을 할 수 있습니다.

처음 소유 확인한 사이트의 경우에는 아래처럼 진행이 되지 않을 수 있습니다. 그러나 약 10분~30분 정도 시간이 지난 후면 최적화 작업을 진행할 수 있습니다.

 

 

 

네이버 사이트 최적화

네이버에서 크게 작업해야 할 사항은 아래의 4가지 입니다.

유형 설명
검색로봇수집 robots.txt, RSS, 사이트맵 등록 여부
콘텐츠 관리 사이트의 meta 태그 등록 여부
사이트 구조 HTML 태그 규약 여부, 모바일 반응형 웹 적용여부, 링크
사이트 활성화 웹 사이트 수집 및 페이지 확산 여부



다른 블로그에도 검색로봇수집에 관련된 내용의 좋은 정보들은 많이 작성되어 있습니다.
저는 직업이 프론트엔드 개발자 이어서 그 외 나머지에 대해서도 상세히 다룰 예정입니다.

 

 

 

 

 

 

검색로봇수집 - robots.txt

robots.txt 파일은 정확히는 로봇 배제 표준이라는 용어로 사용합니다. 웹 사이트에 로봇이 접근하는 것을 방지하는 규약으로 약속된 내용입니다. 가끔 몇몇 사이트들은 웹 크롤링이 되지 않는 것을 원하는 사이트들도 있으나 우리는 블로그가 검색에 노출되기를 희망합니다.

 

티스토리를 사용하신다면 robots.txt 파일은 딱히 건드릴 필요가 없습니다. 이미 카카오에서 메인 도메인, 서브 도메인에 작업을 전부 해주었습니다.

 

본인의 URL에 /robots.txt 입력하시면 확인 사항을 볼 수 있습니다. (ex. https://realmojo.tistory.com/robots.txt)

 

만약 다른 사이트를 직업 운영하고 있으시다면 아래의 파일을 다운로드하여서 루트 폴더에 넣어주시면 됩니다. 네이버 웹 사이트에서 생성한 파일이니 안심하셔도 됩니다. 직접 들어가셔서 받으셔도 됩니다.

 

 

 

검색로봇수집 - RSS 등록

RSS(Rich Site Summary)의 뜻으로 사이트에 등록되어 있는 내용을 외부로 전달하기 위한 내용입니다. 사이트에 바뀐 내용이 있을 경우 RSS에 정보를 텍스트로 간단하게 알려주는 것입니다.

 

네이버에 티스토리 RSS 등록하는 법은 아주 간단합니다. 티스토리에서의 rss 또한 티스토리에서 제공을 해주기 때문에 웹 마스터 도구에 등록만 해주면 끝입니다.

 

 

 

확인 버튼을 눌러주면 2~3초 이내로 등록이 됩니다.

 

 

 

 

 

 

검색로봇수집 - 사이트맵(sitemap.xml) 등록

사이트맵이란 검색로봇에게 사이트 내에 수집되어야 할 페이지들을 알려주기 위한 규약입니다. 사이트 맵을 활용 하면 검색로봇이 해석할 수 있도록 알려주기 때문에 우리가 작성한 콘텐츠를 수집하는데 도움을 더 줍니다.

 

사이트맵 등록은 여러 과정을 좀 거쳐주어야 합니다. 해외 사이트에서 sitemap을 생성해서 파일을 티스토리에 직접 등록해주고 네이버에도 등록해주어야 합니다.

 

우리는 아래의 사이트에서 만들겠습니다. 여러 사이트를 시도해봤지만 아래의 사이트가 가장 효율적입니다.

https://www.xml-sitemaps.com/

 

위의 링크로 접속하셔서 본인의 블로그를 입력하고 More Options를 클릭하여 아래의 값을 Always로 변경을 합니다. 페이지의 변경 주기를 항상 하겠다는 옵션입니다.

 

 

 

시작을 하면 사이트맵이 생성이 됩니다. 본인의 블로그에 글의 개수에 따라 시간은 다르겠지만 약 5분~10분 정도 걸립니다

 

 

완료가 되면 VIEW SITEMAP DETAILS를 클릭하고

 

 

DOWNLOAD YOUR XML SITEMAP FILE을 클릭하여 파일을 받아줍니다.

 

 

 

 

 

 

 

 

 

사이트맵 티스토리에 등록하기

생성된 파일을 티스토리에 등록해주어야 합니다.

파일을 업로드하기 위해서 티스토리 이전의 에디터를 사용해야 합니다. 관리자 메뉴에서 [콘텐츠 - 설정 - 새로운 글쓰기 사용하지 않습니다 - 변경사항 저장]을 해주어서 이전에 에디터로 이동합니다.

 

 

 

이전에 에디터로 글을 작성합니다. 다운로드한 사이트맵의 파일을 파일 버튼을 눌러 업로드해주고 비공개로 올려줍니다. 비공개로 올리면 글은 보이지 않지만 사이트맵의 링크는 접근이 가능합니다. 글을 발행해 줍니다.

 

 

 

발행한 글에 들어가서 사이트맵의 링크를 가져와야 합니다. 우클릭으로 링크 주소 복사를 합니다. 복사한 링크에서 우리는 /attachment/cfile1.uf@XXXXXXXXXXXXXXXXXXXXX.xml 부분만 다시 복사합니다. 

https://realmojo.tistory.com/attachment/cfile1.uf@XXXXXXXXXXXXXXXXXXXXXXx.xml

 

 

 

 

 

웹 마스터 도구에 사이트맵 등록

위에서 복사한 URL을 웹 마스터 도구에 사이트맵 나머지 URL 주소를 복사 후 확인 버튼을 눌러 등록을 하면 완료가 됩니다.

 

 

 

웹 페이지 최적화 확인

웹 페이지 최적화 탭에 들어가서 확인 버튼을 눌러 최적화가 되었는지 확인해볼 수 있습니다. 

 

 

검색 로봇 수집

검색로봇수집은 시간이 좀 많이 걸립니다. 1일 이상 소요될 수도 있으니 느긋하게 기다리시길 바랍니다.
검색 로봇 수집이 최고예요로 변경되었습니다.

 

robots.txt, RSS, 사이트맵을 등록하면서 블로그 최적화하는 방법을 알아보았습니다. 이 외에도 콘텐츠 관리, 사이트 구조, 사이트 활성화 등 작업을 해주어야 합니다.
