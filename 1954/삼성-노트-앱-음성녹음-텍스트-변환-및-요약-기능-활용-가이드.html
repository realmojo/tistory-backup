삼성 노트 앱에서 음성녹음 시작하기

삼성 노트 앱에서 음성녹음을 시작하는 과정은 직관적이고 간단합니다. 먼저 노트 앱을 실행한 후 우측 하단의 연필 모양 아이콘을 눌러 새 노트를 생성합니다. 상단의 플러스 아이콘을 클릭하면 다양한 옵션 중 '음성녹음'을 선택할 수 있습니다. 녹음 버튼을 누르는 즉시 음성녹음이 자동으로 시작되며, 동시에 S펜이나 손가락으로 메모를 작성할 수 있어 음성과 텍스트가 동기화됩니다. 녹음을 중지하려면 타이머 좌측의 정지 버튼을 누르면 되고, 별도의 저장 과정 없이 뒤로가기만 하면 자동으로 저장됩니다.

갤럭시 AI 텍스트 변환 기능 활용법

녹음이 완료된 후 가장 중요한 단계는 음성을 텍스트로 변환하는 과정입니다. 노트 상단의 녹음 아이콘을 클릭하면 저장된 녹음 파일들을 확인할 수 있고, T자 모양의 아이콘을 눌러 텍스트 변환을 실행합니다.

  • 발화자 인식: 여러 명이 대화할 경우 자동으로 발화자를 구분하여 'Speaker 1', 'Speaker 2' 형태로 표시
  • 언어 선택: 30개 이상의 언어를 지원하며 한국어, 영어, 중국어, 일본어 등 주요 언어 완벽 지원
  • 실시간 동기화: 재생 시 해당 시점의 텍스트가 하이라이트되어 음성과 텍스트 매칭 확인 가능
  • 편집 기능: 변환된 텍스트는 선택, 복사, 삭제 등 다양한 편집 작업 가능
