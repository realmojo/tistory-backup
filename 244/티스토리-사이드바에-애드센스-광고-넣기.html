2020-04-24 16:26:23

 

 

오늘은 티스토리
사이드바 메뉴에 애드센스를
추가하는 법을
공유 드리겠습니다.

 

우선 관리자 화면에
[플러그인 - 배너 출력] 

사용 할 수 있도록
적용해 주셔야 합니다.

 

 

배너출력을 클릭하면
아래의 이미지 처럼
적용을 눌러 사용할 수 있게 해줍니다.

 

 

 

 

 

 

 

 

 

 

 

 

 

그리고 나서 

[꾸미기 - 사이드바]
탭에 들어가셔서 보면

[플러그인] 이미지 배너출력
[플러그인] HTML 배너출력

2개 항목이 추가가 됩니다!

 

 

HTML 배너출력의
+ 버튼을 클릭 하면

오른쪽 메뉴에
하나씩 생성이 됩니다.

드래그 해주셔서
원하는 위치에
적절히 배치를 해주시구요

 

 

 

 

 

 

 

 

 

마우스를 오버하면
편집 버튼이 나옵니다. 

클릭!

 

 

 

HTML의 배너출력에

이름 
HTML소스(구글 광고코드)

를 삽입하면 됩니다!

 

 

구글 광고코드는 가급적이면
디스플레이 광고를
사용할 것을 권유 드립니다

왜냐하면 크기에 맞게
반응형으로 잘 적용이 됩니다
