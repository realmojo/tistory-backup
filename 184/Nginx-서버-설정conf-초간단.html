Nginx Conf 파일 설정


기본적으로는 nginx.conf 파일을 설정해야 하지만 해당 경로 안에
Include 경로가 있기 때문에 보통은 다른 파일을 설정 한다.

 server {

        listen 80;  // 80 번 포트를 연다

        location / {  //  루트 경로로 들어왔을 때 이동을 함

                proxy_pass http://127.0.0.1:8000;   // 80번으로 들어오면 8000번 포트로 이동을 하겠다고 선언

                proxy_http_version 1.1;

                proxy_set_header Connection "";

        }


        location ~ ^/static/ {

                root /root/static;  //  /static 경로로 들어온 거는 해당 경로로 포워딩

        }

}



프록시 패스를 사용하지 않은 경우엔
root 변수를 이용하여 경로를 지정해 주면 된다.
