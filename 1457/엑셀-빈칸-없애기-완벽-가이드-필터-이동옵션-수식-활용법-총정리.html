필터 기능으로 빈칸 없애기

엑셀의 필터 기능은 가장 직관적이고 빠른 방법입니다. 데이터 범위를 선택한 후 상단 메뉴에서 '데이터' 탭의 '필터'를 클릭합니다. 열 제목 옆에 나타나는 ▼ 필터 버튼을 클릭하여 나열된 값의 항목들 체크박스를 모두 해제하고 '필드값 없음'에만 체크합니다. 이후 빈칸이 있는 모든 행을 드래그하여 Ctrl + - 키로 삭제하면 됩니다. 필터를 해제하면 엑셀 빈칸이 제거된 깔끔한 데이터만 남게 됩니다.

  • 필터는 행 단위로 빈칸이 있을 때 가장 효과적입니다
  • 대용량 데이터에서도 빠른 속도로 처리 가능합니다
  • 실수로 중요한 데이터를 삭제할 위험이 적습니다

이동 옵션으로 빈셀 선택하기

F5 키 또는 Ctrl + G 키를 누르면 나타나는 '이동' 창을 통해 빈셀만 정확하게 선택할 수 있습니다. '옵션' 화면에서 '빈 셀'에 체크 후 확인을 누르면 빈셀만 선택됩니다. 선택된 빈칸에서 Ctrl + - 키를 눌러 행 전체 삭제 또는 셀 이동을 선택할 수 있습니다. 이 방법은 특정 범위 내에서 빈셀만 골라내어 삭제할 때 매우 유용하며, 엑셀 빈칸 없애기의 핵심 기능으로 활용됩니다.

다만 왼쪽이나 위로 밀려서 삭제되기 때문에 정렬이 흐트러질 수 있어 주의가 필요합니다. 구조화된 데이터에서는 행 단위나 열 단위로 삭제하는 것이 안전합니다.

  • 수식을 이용하면 원본 데이터를 보존할 수 있습니다
  • 대용량 데이터에서도 자동화된 처리가 가능합니다
