자바스크립트만 사용하여 타입스크립트 처럼 보이기 위한 코딩 가이드

/** 을 이용하여 @param {type} 변수 를 작성해주면 된다. 

/**
 *
 * @param {number} a
 * @param {number} b
 */
const add = (a, b) => {
  return a + b;
};
add(a, b);

 

// @ts-check 를 작성해주면 타입스크립트와 비슷한 검증을 할 수 있다.
